PyHelpers by @macwinnie

This is the documentation overview for PyHelpers project by @macwinnie.

To install this package, please run:

pip install macwinnie-pyhelpers

Examples of usage:

Working with python history in Python shell:

from macwinnie_pyhelpers.PyHistory import printHistory
saveHistory("path/to/historyfile.txt")
printHistory()

Handling CSV data

from macwinnie_pyhelpers.CSV import CSV
csv = CSV()
csv.loadData([
    {
        "col1": "val_row1_col1",
        "col2": "val_row1_col2",
        "col3": "val_row1_col3",
        "col4": "val_row1_col4",
    },
    {
        "col1": "val_row2_col1",
        "col2": "val_row2_col2",
        "col3": "val_row2_col3",
        "col4": "val_row2_col4",
    }
], kvDict=True)
print(bool(len(csv) == 2)) # == True

Doing version checks with sentential logic

import macwinnie_pyhelpers.SententialLogic as SL
from macwinnie_pyhelpers.Version import Version
SL.Sentence("1.2.9 >= 1.2.4 <= 1.2.5", interpretingExample=Version("1.2.3")) # == True

Working with Prometheus like metrics

from macwinnie_pyhelpers import Metrics

mc = Metrics.MetricsCollection()
mc.setHelp('name_of_metric', 'information about metric interpretation')
mc.setType('name_of_metric', 'gauge')
for information in list_of_data:
    labels = {} # to be filled from information
    mc.addMetric('name_of_metric', value=information.value, labels=labels)

PyHelpers by @macwinnie

This is the documentation overview for PyHelpers project by @macwinnie.

To install this package, please run:

                pip install macwinnie-pyhelpers

Usage could then e.g. be: 

                from macwinnie_pyhelpers.PyHistory import PyHistory
                history = PyHistory()
                history.print()

                from macwinnie_pyhelpers import CSV, Browser
                browser = Browser.Browser()
